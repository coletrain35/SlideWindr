Keyboard Shortcuts

Master these shortcuts to present like a pro:

  • B or C - Toggle chalkboard/drawing mode
  • S - Open speaker notes view
  • F - Enter fullscreen mode
  • ESC - Slide overview mode
  • ? - Show all keyboard shortcuts
  • Arrow keys - Navigate slides
